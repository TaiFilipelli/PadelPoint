# Proyecto Final Laboratorio 4: Padel Point

### Proyecto FullStack completo con el objetivo de brindarle a un negocio local de paletas de padel su propio e-commerce. Todo lo descrito y detallado a continuación refiere a la parte **Frontend** del proyecto. El repositorio de la parte Backend, asi como su respectiva documentación, se pueden encontrar <a href="https://www.example.com" target="_blank">aquí</a>.

### Nuestro objetivo es el de proporcionarle a este creciente negocio de paletas y articulos de padel un e-commerce atractivo, útil, totalmente funcional y que se pueda mantener fácilmente con el tiempo. En el lado del Frontend, nuestro más grande objetivo es el de poder administrar los datos que vienen del Backend de la forma más ordenada, correcta y bonita posible. Como en todo proyecto Frontend, la UX-UI es clave, y queremos que cualquier persona que ingrese a comprarse cualquier producto no tenga malas experiencias.

## Tecnologias usadas para esta App:

- Tooling: VITE
- Framework: React
- Estilos: Tailwindcss
- Enrutamiento: Wouter
- Local Storage: Zustand
- Iconos: Phospor Icons
- Algunos componentes son de Keep React
