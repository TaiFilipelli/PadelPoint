# Proyecto Final Laboratorio 4: Padel Point

## Proyecto FullStack completo con el objetivo de brindarle a un negocio local de paletas de padel su propio ecommerce. Todo lo descrito y detallado a continuación refiere a la parte **Frontend** del proyecto. El repositorio de la parte Backend, asi como su respectiva documentación, se pueden encontrar <a href="https://www.example.com" target="_blank">aquí</a>.

### Nuestro objetivo es el de proporcionarle a este creciente negocio de paletas y articulos de padel un ecommerce atractivo, útil, totalmente funcional y que se pueda mantener por el tiempo.
